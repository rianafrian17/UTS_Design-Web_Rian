4 Jelaskan pendapat anda mengenai perkembangan HTML mulai dari versi 1 sampai saat ini ! Tentu perkembangan HTML dari versi 1 hingga saat ini telah sangat signifikan, HTML versi 1, yang dirilis pada tahun 1991, merupakan versi pertama dari bahasa markup tersebut. sejak itu Html telah mengalami beberapa revisi besar, seperti HTML 2,0, HTML 3.2 , HTML 4.01, XHTML1.0, HTML 5 dan yang terbaru, HTML 6. Setiap versi memiliki peningkatan signifikan dalam kemampuan, struktur, dan fungsionalitasnya termasuk penambahan elemen dan atribut baru.
